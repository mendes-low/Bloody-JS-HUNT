How the game works

  • Move with the arrow keys. WASD works too.
  • The host controls the Bloody JS icon and hunts the students.
  • If the host catches a student, that student gets a random multiple-choice question.
  • Correct answer: the host becomes weaker for 5 seconds.
  • Wrong answer: the student is eaten and becomes a spectator.
